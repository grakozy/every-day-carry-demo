# EDC Companion

A minimal MVP web app that suggests items to carry based on your planned activities for the day.

## Features

- Simple form-based interface
- Activity-based item suggestions
- Clean, modern UI
- Responsive design

## Usage

1. Open `index.html` in your web browser
2. Enter an activity (e.g., "gym", "office", "hiking")
3. Click "Get Suggestions" to see recommended items

## Supported Activities

The app currently supports suggestions for:
- gym
- office
- hiking
- travel
- beach
- camping
- running
- shopping
- coffee
- meeting

## Setup for GitHub and CodeRabbit

1. **Initialize Git repository:**
   ```bash
   git init
   git add .
   git commit -m "Initial commit: EDC Companion MVP"
   ```

2. **Create a GitHub repository:**
   - Go to GitHub and create a new repository
   - Don't initialize it with a README (since you already have one)

3. **Push to GitHub:**
   ```bash
   git remote add origin <your-repo-url>
   git branch -M main
   git push -u origin main
   ```

4. **Set up CodeRabbit:**
   - Go to [CodeRabbit](https://coderabbit.ai) and sign in
   - Connect your GitHub repository
   - CodeRabbit will automatically detect the `.coderabbit.yaml` file
   - Configure your review preferences in the CodeRabbit dashboard
   - CodeRabbit will start reviewing your code on pull requests

## Files

- `index.html` - Main HTML structure
- `style.css` - Styling
- `script.js` - JavaScript logic for activity suggestions
- `.coderabbit.yaml` - CodeRabbit configuration placeholder
